# Titanium Stripe

Use the native Stripe SDK's (iOS/Android) in Titanium!

## Requirements

- [x] Titanium SDK 9.2.0+
- [x] iOS 12+
- [x] Android 5+
- [x] A server to generate the ephemeral key (e.g. with Node.js or PHP)

See the [app.js](./example/app.js) for a detailed example.
